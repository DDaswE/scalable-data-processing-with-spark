# Scalable Data Processing with Spark

Spark-based data processing exercises covering integer counting, salary analysis, and optimized word-frequency analysis.

## Project summary

This project shows how Spark can be used for scalable batch analysis across both structured and unstructured data. The work includes RDD transformations, DataFrame aggregation, broadcast variables, partition-level computation, and simple statistical visualization.

## Problem

The assignment explores three practical questions:

1. How can distributed computation count and classify large numeric streams efficiently?
2. How can Spark be used to summarize salary data across departments?
3. How can MapReduce-style processing be optimized for text frequency analysis?

## Data

- `integer.txt` - integer stream used for odd/even counting
- `salary.txt` - department-salary records used for statistical analysis
- `shakespeare-1.txt` - text corpus used for word-frequency experiments

## Cloud / big data techniques

- PySpark RDD transformations and actions
- `mapPartitions` for partition-local aggregation
- `reduceByKey` for distributed counting
- Spark DataFrame aggregation
- broadcast variables for faster lookup in targeted word counting
- basic matplotlib/seaborn visualization for result communication

## Key outputs

- odd/even counts from the integer stream
- department-level salary summary statistics
- box plot and pie chart for salary distribution
- optimized target-word counting with broadcast variables
- full word-frequency distribution and top-word plot

## Repository structure

| File | Role |
| --- | --- |
| `num_counting.py` | Counts odd and even integers with Spark RDD operations |
| `salary_counting.py` | Computes salary statistics by department and generates visualizations |
| `mapperreducer.py` | Demonstrates optimized targeted word counting with broadcast variables |
| `Frequency_Distribution.py` | Runs full word-frequency analysis and frequency-distribution reporting |
| `integer.txt` | Input data for the odd/even counting task |
| `salary.txt` | Input data for the salary analysis task |
| `shakespeare-1.txt` | Input text corpus for the word-frequency tasks |
| `*.png` | Result visualizations generated by the scripts |

## Notes

All scripts load local files from this repository directly, so the project can be opened or copied to another machine without editing hard-coded personal paths.
